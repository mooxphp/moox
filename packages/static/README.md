# Moox Static

Static is a lean Moox Entity for translatable reference data without draft, publishing, or soft-delete workflow.

## Features

- Astrotomic translations (`common_name`, `description`)
- Per-locale admin UI with language selector
- Config-driven Filament resource labels and tabs
- Shared codelist resource helpers for `moox/data` consumers
- Lean `BaseStatic*` stack from `moox/core`

## Requirements

See [Requirements](https://github.com/mooxphp/moox/blob/main/docs/Requirements.md).

## Installation

```bash
composer require moox/static
php artisan moox:install
```
