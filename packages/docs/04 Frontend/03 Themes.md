# Themes introduction

Moox themes define the frontend look and behavior on top of the shared stack.

**Included**

- TailwindCSS
- DaisyUI
- AlpineJS
- Alpine-Ajax
- Motion (option)
- ThreeJS (option)

**Required setup**

- Config (if applicable)
- CSS config with Tailwind and Daisy
- Guest layout
- Welcome page
- Menus (Header, Footer, Social, Meta, User) — stored in DB, typically via seeders

**Optional**

- Custom CSS/JS
- Custom components

Themes such as Base, Accessible, and Featherlight can be installed or used as templates when building an app or bundle.
