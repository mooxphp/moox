# Frontend overview

Moox frontend packages provide a consistent stack and structure for building the public-facing site.

**Stack**

- TailwindCSS
- DaisyUI
- AlpineJS
- Alpine-Ajax
- Motion (optional)
- ThreeJS (optional)

**Provided**

- CSS config (Tailwind and Daisy)
- Guest layout
- Welcome page
- Menus (Header, Footer, Social, Meta, User); menu data is stored in the database (e.g. via seeders)

**Optional**

- Custom CSS/JS
- Custom components

Config and theming are documented in the Themes section.
