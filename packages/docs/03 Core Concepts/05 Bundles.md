# Bundles

Bundles are packages that:

- Require a set of other packages
- Provide the main app configuration
- Wire entities, taxonomies, and modules
- Provide the admin panel for the app

## Moox Base Bundle

Moox Core and User packages with a basic admin panel.

## Moox CMS Bundle

Moox Media, Localization, Data, Page, Category, Tag, Frontend, and a Moox theme.

## Moox Shop Bundle

Moox Media, Localization, Data, Product Group, Product, Cart, Frontend, and a Moox theme.

## Moox Pro Bundle

All packages from Base, CMS, and Shop Bundles, excluding Press and DevOps.

## Moox Press Bundle

Includes a WordPress installation and runs on that database.

## Moox DevOps Bundle

Server package to manage, backup, and restore multiple platforms.
