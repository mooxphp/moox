# Repositories

Moox repositories are the source code repositories of the Moox ecosystem. Public and private repos are hosted on GitHub. A repository can be a

- Composer package for PHP, Laravel, and Filament (most cases)
- Monorepo: `moox` (all FOSS packages) and `pro` (private packages)
- App: `moox.org`, `demo`, `desktop` and `mobile` (private)
- Other (e.g. WordPress plugin, VS Code extension set, `.github`)

Most Repositories represent Packages.
