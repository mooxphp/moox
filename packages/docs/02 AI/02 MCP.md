# MCP

Moox MCP is built with [Laravel MCP](https://laravel.com/ai/mcp) and currently in development.

Following Servers will be added:

## Laravel

We require Laravel Boost, so the default Servers for Laravel, Herd, Livewire, Pint and Pest are already in place, see https://laravel.com/docs/12.x/boost#available-ai-guidelines.

## Moox

We currently work on the implementation of the Moox MCP Server.

## Filament

We recommend https://packagist.org/packages/kirschbaum-development/laravel-loop-filament

## Tailwind

We require Laravel Boost, so Tailwind is already in place, otherwise use https://www.npmjs.com/package/tailwindcss-mcp-server

## Motion

We recommend https://github.com/abhishekrajpurohit/motion-dev-mcp (free) or https://motion.dev/docs/studio-ai-context (paid).

## DaisyUI

We recommend https://github.com/birdseyevue/daisyui-mcp (free) or https://daisyui.com/blueprint/ (paid).

## AlpineJS

There are no known MCP servers for [AlpineJS](https://github.com/alpinejs/alpine) or [Alpine Ajax](https://github.com/imacrayon/alpine-ajax) right now.
