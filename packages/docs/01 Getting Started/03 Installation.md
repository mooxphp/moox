# Installation

Install a package with Composer, then run the Moox Installer:

```bash
composer require moox/core
php artisan moox:install
```

If you are unsure which package to use, start with Moox Core and let the Moox Installer guide you.

The Moox Installer handles installing, updating, upgrading, and uninstalling. You do not need to set up Filament, Livewire, or Tailwind manually.
