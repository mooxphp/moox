# Introduction

Moox is an ecosystem of [Laravel packages](https://www.laravel.com) and [Filament plugins](https://filamentphp.com). With Moox you can rapidly build:

- A Laravel app with admin panel
- A CMS platform with frontend or headless (CMS Bundle)
- A web shop or commerce portal (Shop or Commerce Bundles)
- A Laravel app with WordPress behind (Press Bundle)
- A DevOps platform with central backup (DevOps Bundle)

Moox is free and Open Source. Paid packages are in private repositories, named `Pro`.
