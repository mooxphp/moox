# Translation

Moox supports translatable content and configuration. Use language files and the `trans//` config values (e.g. in resources and taxonomies) so labels and content can be translated. Localization is provided by Moox Localization and configured in Bundles.

We use Weblate to translate our packages. Help wanted.
