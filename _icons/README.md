# TallUI Icons

Iconsets for developers. No dependency on other TallUI packages. Compatible with Blade UI Kit. Use them as you want.

## Blade Icons

-   [TallUI Brand Icons](./tallui-brand-icons/README.md) - Brands based on Simple Icons, including colors
-   [TallUI Laravel Icons](./tallui-laravel-icons/README.md) - A set of SVGs from the Laravel Ecosystem

## Country Flags

-   [TallUI Flags Origin](./tallui-flags-origin/README.md) - Country flags Original format
-   [TallUI Flags Rectangle](./tallui-flags-rectangle/README.md) - Country flags rectangle
-   [TallUI Flags Round](./tallui-flags-round/README.md) - Country flags rounded style
-   [TallUI Flags Square](./tallui-flags-square/README.md) - Country flags Square
